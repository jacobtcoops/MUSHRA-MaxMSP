Introduction

Thank you for agreeing to take part in this listening test. The test should take 30-40 minutes to complete.

Please make sure before you begin the test, please make sure that you have read the information sheet and signed the consent form.

This is a modified MUSHRA test:

  • The reference (marked ‘R’) exemplifies the signal characteristics of the sources being rendered as is not a reference for how the spatial rendering should sound like
  • It is not required to set the score of any item to 100%

In each trial, you will be asked to grade the overall listening experience, taking into account the following attributes:

  • Naturalness of the sound
  • How accurate the location of an object is
  • How point-like an object is (all objects in all test cases are rendered as point sources)
  • If front-back and up-down confusion occurs
  • Smoothness of a moving sound (where appropriate)

Each trial refers to one of 8 ‘items’ - the item number will be displayed on screen and corresponds to one of the items described in this document. Please ensure you read the description before completing each trial.

Before You Begin

  1. Please write the last 3 digits from your phone number and the last three digits of your postcode on the post it note provided.
  1. Please click the ‘R’ button and set the reference audio level on the audio interface. This must remain constant throughout the duration of the listening test.
  1. Please familiarise yourself with the test interface and let the test setter know once you have done so.

If you have any questions about the above, please ask.

Item 6: Moving Tambourine

  • The tambourine remains at a fixed distance from the listener
  • The movement completes two cycles, starting from upper elevation 50° to lower elevation -50° and then back up to 50°

Item 7: Moving Pink Noise

  • Pink noise moving in a diamond in front of the listener
  • The movement starts at the top position completes one cycle during the length of the item

Item 8: Moving Tambourine II

  • Tambourine moves in a vertical circle around the listener
  • The movement starts in front of the listener and completes one cycle during the length of the item
